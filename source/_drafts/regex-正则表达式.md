# Regex 语法列表

|表达式|含义|备注|
|-|-|-|
|`.`|表示一个任意字符||
|`+`|字符至少出现一次||
|`*`|字符出现 0 次或任意次|`.*` 等价于通配符 `*`|
|`?`|字符出现 0 次或 1 次|`.?` 等价于通配符 `?`|
|`|`|||
|`()`|||
|`[]`|||
|`{n}`|字符正好出现 n 次||
|`{n,}`|字符至少出现 n 次||
|`{m,n}`|字符出现 m 到 n 次||
|`\s`|任意数量的空白字符|如空格, 换行符等|
|`\S`|任意数量的可见字符|如普通字母, 汉字等|
|`\b`|匹配一个单词边界|如 `This is a high tower`, `hi\b` 将会匹配 `high` 中的 `hi`, 而 `This` 中的 `hi` 不会匹配|
|`\B`|匹配一个非单词边界||
|`\`|转义字符||
