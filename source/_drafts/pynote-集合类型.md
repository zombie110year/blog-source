## 列表-list

Python 中使用 `[]` 包括多个元素来表示 list.

```py
list = ["高数", 59, "挂科", 0.99] # 将 list 类型的对象赋值给一个命名为 list 的变量.
print(list)

# 输出信息:
['高数', 59, '挂科', 0.99]        # list 中可以包含任意类型的对象, 且有序排列.
```
